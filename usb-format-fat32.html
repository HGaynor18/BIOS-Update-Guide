How to Format a USB Drive to FAT32

What is FAT32?

FAT32 is a file system that's compatible with many operating systems, including Windows, macOS, Linux, and most gaming consoles. Formatting a USB drive to FAT32 is often necessary for compatibility with older systems or certain devices like BIOS updates, cameras, or video game consoles.

Step-by-Step Guide to Format a USB Drive to FAT32 on Windows

  1. Insert the USB drive into your computer.
  2. Open File Explorer and locate the USB drive in the "This PC" or "My Computer" section.
  3. Right-click on the USB drive and select Format from the context menu.
  4. A new window will open. Under the File System dropdown, select FAT32.
  5. Ensure the checkbox for Quick Format is checked (this speeds up the process).
  6. Click Start to begin formatting the drive.
  7. When the process is complete, a confirmation message will appear. Click OK to close the window.
Important: Formatting will erase all data on the USB drive. Be sure to back up any important files before proceeding.

Formatting a USB Drive to FAT32 on macOS

  1. Insert the USB drive into your Mac.
  2. Open Disk Utility (you can find it using Spotlight search).
  3. Select the USB drive from the list of drives on the left-hand side.
  4. Click the Erase button at the top of the window.
  5. In the pop-up window, select MS-DOS (FAT) under the Format dropdown (this is the FAT32 equivalent on macOS).
  6. Click Erase to begin formatting.
  7. Once completed, the drive will be formatted to FAT32, and you can safely eject it.

Formatting a USB Drive Using Command Prompt (Windows)

If the USB drive is larger than 32GB, Windows won't let you format it to FAT32 using the default File Explorer tool. In this case, you can use the Command Prompt.

  1. Type cmd in the Windows search bar, right-click Command Prompt, and select Run as administrator.
  2. In the command prompt, type: format /FS:FAT32 X: (replace X: with the correct letter of your USB drive).
  3. Press Enter to start the formatting process.
Note: This method may take a long time for large drives, as the Command Prompt does not have a quick format option.
